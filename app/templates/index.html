通知渠道说明

                            

                                

                                    
企业微信机器人

                                    
适合企业微信群通知，配置机器人 Key 即可。

通过企业自建应用推送，需 corpid/corpsecret/touser/agentid，可选 media_id。

                                

                                

                                    
Telegram

                                    
使用 Bot Token 与用户ID，支持可选代理。

                                

                                

                                    
邮件 (SMTP)

                                    
配置服务器、邮箱与密码，支持 SSL 与端口。

                                

                                

                                    
iGot / Server酱 / 飞书 / 钉钉

                                    
常用聚合与办公通知渠道，分别填写对应的 Key 或 Token 即可。

                                

                                

                                    
自定义 Webhook

                                    
支持 POST/GET，自定义 Header 与 Body（含占位符 $title/$content）。
